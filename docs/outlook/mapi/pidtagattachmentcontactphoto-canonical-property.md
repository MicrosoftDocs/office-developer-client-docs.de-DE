# <a name="pidtagattachmentcontactphoto-canonical-property"></a>PidTagAttachmentContactPhoto (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt das Vorhandensein einer Fotoanlage für einen Kontakt an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_ATTACHMENT_CONTACTPHOTO  <br/> |
|Kennung:  <br/> |0x7FFF  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Nachrichtenanlage  <br/> |
   
## <a name="remarks"></a>Hinweise

Der Wert dieser Eigenschaft muss auf TRUE festgelegt werden, und es sollte nur eine Anlage für ein bestimmtes Contact-Objekt sein.
