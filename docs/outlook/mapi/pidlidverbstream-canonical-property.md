# <a name="pidlidverbstream-canonical-property"></a>PidLidVerbStream (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, welche Abstimmungsantworten der Benutzer als Antwort auf die Nachricht geben kann.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidVerbStream  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Lange ID (LID):  <br/> |0x00008520  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Laufzeitkonfiguration  <br/> |
