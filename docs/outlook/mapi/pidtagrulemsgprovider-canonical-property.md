# <a name="pidtagrulemsgprovider-canonical-property"></a>PidTagRuleMsgProvider (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält zusätzliche Informationen zur Clientanwendung, die eine erweiterte Regel besitzt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RULE_MSG_PROVIDER  <br/> |
|Kennung:  <br/> |0x65EB  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Regeln  <br/> |
