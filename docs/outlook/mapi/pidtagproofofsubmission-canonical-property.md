# <a name="pidtagproofofsubmission-canonical-property"></a>PidTagProofOfSubmission (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Diese Eigenschaft enthält TRUE, wenn ein Nachrichtensender den Nachweis anfordert, dass das Nachrichtenübertragungssystem eine Nachricht zur Zustellung an den ursprünglich beabsichtigten Empfänger übermittelt hat.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_PROOF_OF_SUBMISSION  <br/> |
|Kennung:  <br/> |0x0E26  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI Nicht durchlässig  <br/> |
