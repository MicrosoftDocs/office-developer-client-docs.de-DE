# <a name="pidtagfreebusypublishstart-canonical-property"></a>PidTagFreeBusyPublishStart (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Startzeit des Veröffentlichungsbereichs.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_FREEBUSY_PUBLISH_START  <br/> |
|Kennung:  <br/> |0x6847  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Frei/Gebucht  <br/> |
   
## <a name="remarks"></a>Hinweise

Der Wert für diese Eigenschaft ist die Anzahl von Minuten seit Mitternacht, 1. Januar 1601 in koordinierter Weltzeit (UTC).
