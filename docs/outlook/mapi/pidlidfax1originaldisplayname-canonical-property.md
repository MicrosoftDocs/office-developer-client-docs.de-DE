# <a name="pidlidfax1originaldisplayname-canonical-property"></a>PidLidFax1OriginalDisplayName (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den ursprünglichen Anzeigenamen der Geschäftsfaxadresse des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFax1OriginalDisplayName  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Lange ID (LID):  <br/> |0x000080B4  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft muss, falls vorhanden, auf denselben Wert wie die **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)) festgelegt werden.
