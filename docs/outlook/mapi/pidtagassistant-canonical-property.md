# <a name="pidtagassistant-canonical-property"></a>PidTagAssistant (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Namen des Administrativen Assistenten des Empfängers. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_ASSISTANT, PR_ASSISTANT_A, PR_ASSISTANT_W  <br/> |
|Kennung:  <br/> |0x3A30  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften stellen Identifikations- und Zugriffsinformationen für einen Empfänger zur Verfügung. Sie werden vom Empfänger und seiner Organisation definiert.
