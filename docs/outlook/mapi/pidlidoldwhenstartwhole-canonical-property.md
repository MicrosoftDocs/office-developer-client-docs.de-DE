# <a name="pidlidoldwhenstartwhole-canonical-property"></a>PidLidOldWhenStartWhole (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den ursprünglichen Wert der **eigenschaft dispidApptStartWhole** ([PidLidAppointmentStartWhole](pidlidappointmentstartwhole-canonical-property.md)) vor einem Besprechungsupdate an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidOldWhenStartWhole  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Meeting  <br/> |
|Lange ID (LID):  <br/> |0x00000029  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist nicht erforderlich.
