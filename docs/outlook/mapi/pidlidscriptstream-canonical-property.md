# <a name="pidlidscriptstream-canonical-property"></a>PidLidScriptStream (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Represents the code written in Visual Basic Scripting Edition (VBScript) behind the custom form associated with the message.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidScriptStream  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Lange ID (LID):  <br/> |0x00008541  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Laufzeitkonfiguration  <br/> |
