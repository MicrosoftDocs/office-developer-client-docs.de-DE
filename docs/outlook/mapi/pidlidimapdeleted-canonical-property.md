# <a name="pidlidimapdeleted-canonical-property"></a>PidLidImapDeleted (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt Elemente des Internet Mail Access Protocol (IMAP) an, die zum Löschen markiert sind.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidImapDeleted  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Lange ID (LID):  <br/> |0x00008570  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |IMAP  <br/> |
   
## <a name="remarks"></a>Hinweise

Wenn dieser Wert auf einen Wert ungleich Null festgelegt ist, wurde das Element zum Löschen markiert.
