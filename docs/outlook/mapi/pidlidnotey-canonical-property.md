# <a name="pidlidnotey-canonical-property"></a>PidLidNoteY (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den Abstand (in Pixel) vom oberen Rand des Bildschirms an, auf dem eine Benutzeroberfläche eine Notiz anzeigt.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidNoteY  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Note  <br/> |
|Lange ID (LID):  <br/> |0x00008B05  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Sticky Note  <br/> |
