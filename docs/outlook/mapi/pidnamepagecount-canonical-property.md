# <a name="pidnamepagecount-canonical-property"></a>PidNamePageCount (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Seitenanzahl der Datei an, die dem Dokument zugeordnet ist.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftsname:  <br/> |PageCount  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Standard  <br/> |
