# <a name="pidtagkeyword-canonical-property"></a>PidTagKeyword (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält ein Schlüsselwort, das den Empfänger für den Systemadministrator des Empfängers identifiziert.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_KEYWORD, PR_KEYWORD_A, PR_KEYWORD_W  <br/> |
|Kennung:  <br/> |0x3A0B  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften stellen Identifikations- und Zugriffsinformationen für einen Empfänger zur Verfügung. Sie werden vom Empfänger und seiner Organisation definiert.
