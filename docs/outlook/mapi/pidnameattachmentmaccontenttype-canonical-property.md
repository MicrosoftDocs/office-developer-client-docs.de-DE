# <a name="pidnameattachmentmaccontenttype-canonical-property"></a>PidNameAttachmentMacContentType (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält einen [RFC3282] Content-Type-Headerfeldwert für Anlagen, die von Macintosh-basierten E-Mail-Clients verwendet werden.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Attachment  <br/> |
|Eigenschaftsname:  <br/> |AttachmentMacContentType  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Nachrichtenanlage  <br/> |
