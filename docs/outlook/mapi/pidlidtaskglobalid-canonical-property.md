# <a name="pidlidtaskglobalid-canonical-property"></a>PidLidTaskGlobalId (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Sucht einen vorhandenen Vorgang mithilfe einer GUID nach Eingang einer Aufgabenantwort oder Aufgabenaktualisierung.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskGlobalObjId  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Common  <br/> |
|Lange ID (LID):  <br/> |0x00008519  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft wird für nicht zugewiesene Vorgänge nicht zugewiesen.
