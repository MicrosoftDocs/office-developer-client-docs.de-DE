# <a name="pidtagscheduleinfofreebusy-canonical-property"></a>PidTagScheduleInfoFreeBusy (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält veraltete Informationen.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_SCHDINFO_FREEBUSY  <br/> |
|Kennung:  <br/> |0x686C  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Frei/Gebucht  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft sollte nicht festgelegt und ignoriert werden.
