# <a name="pidtagrtfsynctrailingcount-canonical-property"></a>PidTagRtfSyncTrailingCount (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine Anzahl der ignorablen Zeichen, die nach den wichtigen Zeichen der Nachricht angezeigt werden.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_RTF_SYNC_TRAILING_COUNT  <br/> |
|Kennung:  <br/> |0x1011  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |MAPI-Nachricht  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist eine Rich Text Format (RFT)-Hilfseigenschaft. Diese Eigenschaften werden von der [RTFSync-Funktion](rtfsync.md) verwendet und sind nicht für die direkte Verwendung durch Clientanwendungen vorgesehen.
