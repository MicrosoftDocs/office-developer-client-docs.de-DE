# <a name="pidnameapplicationname-canonical-property"></a>PidNameApplicationName (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die Anwendung an, mit der die Datei geöffnet werden kann, die dem Dokument angefügt ist.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftsname:  <br/> |AppName  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Standard  <br/> |
