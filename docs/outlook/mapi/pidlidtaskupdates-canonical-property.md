# <a name="pidlidtaskupdates-canonical-property"></a>PidLidTaskUpdates (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob der Aufgabenbef?hnte zum Senden einer Vorgangsaktualisierung angefordert wurde, wenn sich der zugewiesene Vorgang ändert.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidTaskUpdates  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Task  <br/> |
|Lange ID (LID):  <br/> |0x0000811B  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Aufgabe  <br/> |
