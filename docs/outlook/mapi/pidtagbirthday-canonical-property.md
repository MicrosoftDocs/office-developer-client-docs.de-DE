# <a name="pidtagbirthday-canonical-property"></a>PidTagBirthday (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Geburtstag des Kontakts.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_BIRTHDAY  <br/> |
|Kennung:  <br/> |0x3A42  <br/> |
|Datentyp:  <br/> |PT_SYSTIME  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist eine der Eigenschaften, die Identifikations- und Zugriffsinformationen zu einem Kontakt bieten. Diese Eigenschaften werden vom Benutzer und der Organisation des Benutzers definiert.
