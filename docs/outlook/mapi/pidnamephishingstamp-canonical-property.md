# <a name="pidnamephishingstamp-canonical-property"></a>PidNamePhishingStamp (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Nachrichtenwert, der Nachrichten identifiziert und markiert, bei der es sich wahrscheinlich um Phishing handelt.
  
|||
|:-----|:-----|
|Anzeigenamen:  <br/> |Keine  <br/> |
|Eigenschaftensatz:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Eigenschaftsname:  <br/> |http://schemas.microsoft.com/outlook/phishingstamp  <br/> |
|Datentyp:  <br/> |PT_LONG  <br/> |
|Bereich:  <br/> |Sicheres Messaging  <br/> |
