# <a name="pidtagotheraddresspostofficebox-canonical-property"></a>PidTagOtherAddressPostOfficeBox (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält das Postfach für die andere Adresse eines Kontakts.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_OTHER_ADDRESS_POST_OFFICE_BOX, PR_OTHER_ADDRESS_POST_OFFICE_BOX_A, PR_OTHER_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Kennung:  <br/> |0x3A64  <br/> |
|Datentyp:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Bereich:  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaften werden vom Benutzer oder der Organisation des Benutzers definiert.
