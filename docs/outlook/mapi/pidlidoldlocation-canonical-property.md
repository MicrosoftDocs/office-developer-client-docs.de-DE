# <a name="pidlidoldlocation-canonical-property"></a>PidLidOldLocation (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den ursprünglichen Wert der **eigenschaft dispidLocation** ([PidLidLocation](pidlidlocation-canonical-property.md)) vor einer Besprechungsaktualisierung an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidOldLocation  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Meeting  <br/> |
|Lange ID (LID):  <br/> |0x00000028  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Besprechungen  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft ist nicht erforderlich.
