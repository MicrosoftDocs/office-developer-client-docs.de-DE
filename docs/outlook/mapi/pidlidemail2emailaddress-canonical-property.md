# <a name="pidlidemail2emailaddress-canonical-property"></a>PidLidEmail2EmailAddress (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die zweite E-Mail-Adresse des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidEmail2EmailAddress  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Lange ID (LID):  <br/> |0x00008093  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Der Wert dieser Eigenschaft muss für den für diese E-Mail-Adresse angegebenen Adresstyp geeignet sein.
