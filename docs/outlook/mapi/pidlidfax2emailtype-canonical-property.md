# <a name="pidlidfax2emailtype-canonical-property"></a>PidLidFax2EmailType (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt den E-Mail-Typ der Faxadresse des Kontakts an.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidFax2EmailType  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Lange ID (LID):  <br/> |0x000080C7  <br/> |
|Datentyp:  <br/> |PT_UNICODE  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
