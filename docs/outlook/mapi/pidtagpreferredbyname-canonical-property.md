# <a name="pidtagpreferredbyname-canonical-property"></a>PidTagPreferredByName (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält den Namen, den der Empfänger bevorzugt angerufen werden möchte.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_PREFERRED_BY_NAME, PR_PREFERRED_BY_NAME_A, PR_PREFERRED_BY_NAME_W  <br/> |
|Kennung:  <br/> |0x3A47  <br/> |
|Datentyp:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Bereich:  <br/> |MAPI-E-Mail-Benutzer  <br/> |
