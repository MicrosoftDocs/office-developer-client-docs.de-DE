# <a name="pidtagprocessed-canonical-property"></a>PidTagProcessed (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Wird auf TRUE festgelegt, wenn die Besprechungsanfrage verarbeitet wurde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_PROCESSED  <br/> |
|Kennung:  <br/> |0x7D01  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Kalender  <br/> |
   
## <a name="remarks"></a>Hinweise

Diese Eigenschaft stellt sicher, dass Besprechungsanfragen einmal verarbeitet werden. Der Ersteller der Anforderung sollte diese Eigenschaft auf FALSE festlegen, und der Empfänger sollte sie auf TRUE festlegen, sobald sich die Anforderung im Kalender befindet.
