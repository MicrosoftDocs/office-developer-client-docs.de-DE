<span data-ttu-id="a6bbc-105">Benachrichtigungen in einer Anzeigetabelle werden vom Dienstanbieter gesendet, der für das Erstellen der Anzeigetabelle an MAPI zuständig ist.</span><span class="sxs-lookup"><span data-stu-id="a6bbc-105">Notifications on a display table are sent by the service provider responsible for creating the display table to MAPI.</span></span> <span data-ttu-id="a6bbc-106">MAPI registriert diese Benachrichtigungen, indem die [IMAPITable::Advise-Methode](imapitable-advise.md) einer Anzeigetabelle aufruft und das Table Modified-Ereignis angegeben wird.</span><span class="sxs-lookup"><span data-stu-id="a6bbc-106">MAPI registers for these notifications by calling a display table's [IMAPITable::Advise](imapitable-advise.md) method and specifying the table modified event.</span></span>
