# <a name="pidtagfinderentryid-canonical-property"></a>PidTagFinderEntryId (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält die Eintrags-ID für den Ordner, in dem in der Regel Suchergebnisse erstellt werden.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_FINDER_ENTRYID  <br/> |
|Kennung:  <br/> |0x35E7  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |MAPI-Nachrichtenspeicher  <br/> |
   
## <a name="remarks"></a>Hinweise

Der in dieser Eigenschaft enthaltene Eintragsbezeichner hat das gleiche Format wie die [ENTRYID-Struktur.](entryid.md)
