# <a name="pidlidlogdocumentposted-canonical-property"></a>PidLidLogDocumentPosted (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt an, ob das Dokument während des Journalings per E-Mail gesendet oder in einem Serverordner bereitgestellt wurde.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidLogDocPosted  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Log  <br/> |
|Lange ID (LID):  <br/> |0x00008711  <br/> |
|Datentyp:  <br/> |PT_BOOLEAN  <br/> |
|Bereich:  <br/> |Journal  <br/> |
