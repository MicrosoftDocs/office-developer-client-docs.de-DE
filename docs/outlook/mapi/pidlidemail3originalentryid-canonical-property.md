# <a name="pidlidemail3originalentryid-canonical-property"></a>PidLidEmail3OriginalEntryId (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Gibt die **EntryId des** Objekts an, das der dritten E-Mail-Adresse entspricht. 
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |dispidEmail3OriginalEntryID  <br/> |
|Eigenschaftensatz:  <br/> |PSETID_Address  <br/> |
|Lange ID (LID):  <br/> |0x000080A5  <br/> |
|Datentyp:  <br/> |PT_BINARY  <br/> |
|Bereich:  <br/> |Kontakt  <br/> |
   
## <a name="remarks"></a>Hinweise

Der Wert dieser Eigenschaft muss entweder eine one-off **EntryId** für diese elektronische Adresse oder ein gültiges Adressbuchobjekt **EntryId sein.**
