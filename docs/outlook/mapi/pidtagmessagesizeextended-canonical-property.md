# <a name="pidtagmessagesizeextended-canonical-property"></a>PidTagMessageSizeExtended (kanonische Eigenschaft)

  
  
**Gilt für**: Outlook 2013 | Outlook 2016 
  
Enthält eine ganzzahlige 64-Bit-Version der **PR_MESSAGE_SIZE** ([PidTagMessageSize](pidtagmessagesize-canonical-property.md)) -Eigenschaft.
  
|||
|:-----|:-----|
|Zugeordnete Eigenschaften:  <br/> |PR_MESSAGE_SIZE_EXTENDED  <br/> |
|Kennung:  <br/> |0x0E08  <br/> |
|Datentyp:  <br/> |PT_I8  <br/> |
|Bereich:  <br/> |Allgemeines Messaging  <br/> |
